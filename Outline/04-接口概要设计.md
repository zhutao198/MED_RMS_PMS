## 1. 接口设计规范

### 1.1 RESTful API 通用规范

| 维度 | 规范 |
|------|------|
| **URL风格** | `/api/v1/{module}/{resource}[/{id}][/{sub-resource}]` |
| **HTTP方法** | GET（查询）、POST（创建）、PUT（全量更新）、PATCH（部分更新）、DELETE（删除） |
| **认证方式** | JWT Bearer Token（Authorization头），电子签名场景需额外 OTP 校验 |
| **请求格式** | `Content-Type: application/json` |
| **响应格式** | 统一 JSON 信封：`{code, message, data, timestamp}` |
| **分页参数** | `page`（从1开始）、`size`（默认20，最大100）、`sort`（field,direction） |
| **时间格式** | ISO 8601：`yyyy-MM-dd'T'HH:mm:ss.SSSZ` |
| **ID格式** | 雪花算法生成的Long型，API中以String传输 |
| **版本控制** | URL路径版本 `/api/v1/`，重大变更升级版本号 |

### 1.2 统一响应信封

```json
{
    "code": 200,
    "message": "success",
    "data": { ... },
    "timestamp": "2026-05-22T14:30:00.000+08:00"
}
```

**分页响应：**

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "content": [ ... ],
        "pagination": {
            "page": 1,
            "size": 20,
            "total": 156,
            "totalPages": 8
        }
    },
    "timestamp": "2026-05-22T14:30:00.000+08:00"
}
```

## 3. 内部服务调用接口

### 3.1 模块间同步调用

| 调用方 | 被调用方 | 接口 | 协议 | 说明 |
|--------|----------|------|------|------|
| chg-mgr | e-sign | `POST /internal/esign/intent` | REST | 变更审批触发签名意图 |
| compliance | e-sign | `POST /internal/esign/intent` | REST | 基线锁定触发签名意图 |
| req-mgr | e-sign | `POST /internal/esign/intent` | REST | 评审通过触发签名意图 |
| 各业务模块 | sys-mgr | `GET /internal/users/{id}` | REST | 查询用户基本信息 |
| 各业务模块 | sys-mgr | `GET /internal/permissions/check` | REST | 权限校验（Spring Security共享库间接调用） |

### 3.2 内部接口规范

- **URL前缀**：`/internal/`（不走JWT认证，走服务间认证）
- **认证方式**：服务间共享密钥（Header: `X-Internal-Token`）
- **限流**：内部接口不做限流，但需熔断保护
- **文档**：内部接口不对外暴露，仅在内部分享API文档

## 7. 接口安全规范

### 7.1 认证与授权

| 接口类型 | 认证方式 | 说明 |
|----------|----------|------|
| 前端API | JWT Bearer Token | Access Token 15min + Refresh Token 24h |
| 电子签名API | JWT + OTP + 签名密码 | 三重认证 |
| 内部API | X-Internal-Token | 服务间共享密钥 |
| 第三方回调 | HMAC签名校验 | OA回调需验签 |

### 7.2 数据安全

| 措施 | 说明 |
|------|------|
| HTTPS | 所有接口强制 HTTPS |
| 敏感字段脱敏 | 手机号/邮箱在列表接口中部分隐藏 |
| 请求大小限制 | 默认最大 10MB，文件上传最大 100MB |
| 限流 | 按用户+API粒度限流，默认 100次/分钟 |
| SQL注入防护 | MyBatis-Plus 参数化查询，禁止拼接SQL |
| XSS防护 | 输入内容 HTML 转义存储 |
