## 架构速览

```
技术栈：Spring Boot 3.3.x + Java 17 + MyBatis-Plus + PostgreSQL 16 + Redis 7 + Debezium + MinIO + Vue 3 + Element Plus

架构风格：模块化单体（Spring Modulith），9个限界上下文，单库多Schema隔离

核心设计模式：
  - 闭包表 + CTI 存储需求层级
  - Transactional Outbox + Debezium CDC 跨模块事件
  - 审计日志哈希链（prev_hash/current_hash）
  - JWT 与电子签名分离认证
  - CQRS Lite 仪表盘统计

合规覆盖：IEC 62304 / ISO 13485 / 21 CFR Part 11 / ISO 14971 / NMPA eRPS
```

## 关键约束一览

| 维度 | 约束 |
|------|------|
| 审计日志 | 追加只写，不可UPDATE/DELETE，哈希链保护，永久保留 |
| 电子签名 | 签名密码+OTP双重认证，SHA-256绑定文档哈希，独立于JWT |
| 基线锁定 | 双人签名锁定，锁定后不可修改，解锁需QA经理签名 |
| DCP门控 | 5个阶段门自动校验+人工评审，校验不通过不可进入下一阶段 |
| 权限控制 | RBAC 8类角色，菜单/按钮/API/数据四级权限控制 |
| 并发控制 | 乐观锁（version字段），状态机防跳步 |
| 性能目标 | 常规列表≤500ms，详情≤300ms，可用性≥99.5% |
