**现象**: 后端配置 `context-path: /api`，Controller层也配置了 `/api/soup`，导致实际路径变成 `/api/api/soup`

**建议**: 移除Controller层的 `/api` 前缀，保持路径简洁

## 十二、数据库问题发现

### 问题：Baseline表不存在

**现象**: 创建基线API返回系统异常

**原因**: `compliance_schema.t_baseline` 表在DDL文件中定义但未在数据库中创建

**DDL定义** (init_database.sql):
```sql
CREATE TABLE IF NOT EXISTS compliance_schema.t_baseline (
    id BIGSERIAL PRIMARY KEY,
    baseline_no VARCHAR(50) NOT NULL UNIQUE,
    baseline_name VARCHAR(200) NOT NULL,
    baseline_type VARCHAR(50) NOT NULL,
    project_id BIGINT,
    status VARCHAR(50) DEFAULT 'Draft',
    locked_by BIGINT,
    locked_at TIMESTAMP,
    snapshot_data JSONB,
    is_deleted BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**解决**: 需要执行DDL脚本创建表，或让数据库管理员初始化数据库Schema
